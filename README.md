# Horti-tech
# 📊 Sistema Inteligente de Gestão para HortiFrutis com AIoT

## 📌 Visão Geral
Este projeto propõe o desenvolvimento de um sistema inteligente para gestão de hortifrutis, utilizando **AIoT (Artificial Intelligence of Things)** — a integração entre Inteligência Artificial (IA) e Internet das Coisas (IoT).

O objetivo é reduzir desperdícios, otimizar a gestão de estoque e melhorar a tomada de decisões por meio de dados em tempo real e análises preditivas.

---

## ⚠️ Problema

Hortifrutis lidam diariamente com produtos altamente perecíveis, como frutas, verduras e legumes, que possuem:

- Curto prazo de validade  
- Sensibilidade à temperatura e umidade  
- Alta rotatividade  

Sem um controle eficiente, surgem diversos problemas:

- Falta de monitoramento preciso de estoque  
- Decisões baseadas apenas na experiência  
- Dificuldade em acompanhar validade dos produtos  
- Erros na precificação  
- Falhas na previsão de demanda  

---

## 📉 Impacto no Negócio

Esses desafios geram consequências diretas:

- Aumento do desperdício de alimentos  
- Redução da margem de lucro  
- Compras mal planejadas  
- Perda de competitividade no mercado  

Negócios que não adotam tecnologia acabam ficando atrás de concorrentes mais inovadores.

---

## 🤖 Solução: AIoT

O sistema utiliza o conceito de **AIoT**, combinando:

### 🌐 Internet das Coisas (IoT)
Responsável pela coleta de dados em tempo real através de sensores:

- Temperatura de geladeiras  
- Umidade do ambiente  
- Quantidade de produtos em estoque  

### 🧠 Inteligência Artificial (IA)
Responsável por:

- Analisar dados coletados  
- Identificar padrões  
- Gerar previsões  
- Apoiar decisões estratégicas  

---

## ⚙️ Funcionamento

1. Sensores capturam dados do ambiente e estoque  
2. Dados são enviados via APIs REST para o back-end  
3. Informações são armazenadas em banco de dados  
4. A IA processa os dados e identifica padrões  
5. O sistema gera alertas e recomendações  

### 📌 Exemplo prático:
- Aumento de temperatura em uma geladeira  
- Sistema detecta risco de perda  
- Alerta automático é enviado ao gestor  

---

## 🔮 Inteligência e Automação

O sistema não apenas monitora, mas também age de forma preventiva:

- Sugestão de descontos para produtos próximos do vencimento  
- Alertas em tempo real  
- Recomendações de armazenamento  
- Ajustes na previsão de compras  

### 📊 Análise Preditiva

Com base em:

- Histórico de vendas  
- Dados de estoque  
- Padrões de consumo  

Exemplo:
- Produto com baixa saída em certos dias  
- Sistema reduz sugestão de compra  

---

## 🚀 Benefícios

- Redução significativa do desperdício  
- Aumento da lucratividade  
- Melhor controle de estoque  
- Decisões baseadas em dados  
- Maior eficiência operacional  

---

## 🛠️ Tecnologias Envolvidas

- **Back-end:** APIs REST  
- **Banco de Dados:** Armazenamento estruturado  
- **Front-end:** Interface responsiva (UX/UI)  
- **Mobile:** Gestão em tempo real  
- **IoT:** Sensores físicos  
- **IA:** Modelos de análise e previsão  

---

## 🎯 Conclusão

Este projeto aplica conceitos modernos de desenvolvimento de sistemas e transformação digital para resolver um problema real do mercado.

A integração entre IoT e Inteligência Artificial permite que hortifrutis evoluam de uma gestão reativa para uma gestão **inteligente, automatizada e orientada por dados**.
